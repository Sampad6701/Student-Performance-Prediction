## Problem Statement

The project attempts to answer:

> *Educational institutions aim to identify students at risk of poor academic performance.

This version of project focuses on building a simple and transparent supervised learning pipeline for academic demonstration.

## Current Limitations (V1)

This is a classroom-scale prototype and therefore:

- Uses a small dataset  
- Relies on a single baseline model  
- Limited hyperparameter tuning  
- No deployment pipeline  
- Intended for academic demonstration
